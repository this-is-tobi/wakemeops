    eza

    Modern, maintained replacement for ls, built on exa

    eza uses colours for information by default, helping you distinguish between many types of files, such as whether you are the owner, or in the owning group. It also has extra features not present in the original ls, such as viewing the Git status for a directory, or recursing into directories with a tree view.

    terminal

    https://github.com/eza-community/eza

    arm64 amd64 armhf

    Installation

    1. Add WakeMeOps repository

    curl -sSL https://raw.githubusercontent.com/upciti/wakemeops/main/assets/install_repository | sudo bash

    2. Install eza

    sudo apt install eza

    Snippets

    FROM wakemeops/minideb:bullseye

RUN install_packages \
    eza=0.18.11*

USER 1001

    - name: Install dependencies
  uses: upciti/wakemeops-action@v1
  with:
    packages: |
      eza=0.18.11*
